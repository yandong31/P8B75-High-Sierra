Miscellaneous Information

This tab displays other pertinent CPUID information not falling under the other categories.  The following describes the different items potentially listed in this tab:

Processor Threads
Logical Processors per Core specifies how many logical processors there are in a processor core.
Cores per Physical Package specifies how many microprocessor cores there are in a physical package.

Monitor/MWait Leaf
Smallest Line Size specifies the smallest address line size the MONITOR/MWAIT instructions examine.
Largest Line Size specifies the largest address line size the MONITOR/MWAIT instructions examine.
Interrupts Treated as Break-Event specifies that the processor treats interrupts as a break-event for MWAIT even when interrupts are disabled.

Thermal and Power Management Leaf
Number of Interrupt Thresholds specifies how many interrupt thersholds there are for the digital thermal sensor.
ACNT/MCNT specifies the capability to provide a measure of delivered processor performance (since the last reset of the counters), as a percentage of expected processor performance at the frequency specified in the CPUID Brand String.

Address Sizes
Physical Address specifies the largest physical address the processor is capable of.
Logical Address specifies the largest logical address the processor is capable of.
