General Information

This tab displays general information retrieved from the CPUID instruction.

The processor manufacturer, identifying this as a genuine Intel® processor by returning the non-zero terminated ASCII string 'GenuineIntel'.

The processor brand string, which is a zero terminated ASCII brand identifier string for the system's processor.

The family, model, stepping, and extended family and model number which identify the type of processor this is.  The extended family number only needs to be checked when the family number if 15 and the extended model number only needs to be checked when the family number is either 6 or 15.

The time stamp counter (TSC) frequency, and an indication as to whether this frequency can vary or not.
